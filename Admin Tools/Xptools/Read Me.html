1. How to use RPGWO Level Tester

The purpose of the RPGWO Level Tester is to allow you to test your levels that you create before you submit them to an admin.  It also allows you to test a level and see if it's too hard or too easy.

To install this program, copy it and all it's data files to the directory of rpgwo.

You can use either map files from the RPGWO Edit or the .rsf files from RPGWO Edit.

  In order to load a map into my level tester you have to get a .rsf file.    

To get a .rsf.  First open a map into the rpgwo map editor. Next go to file then dump script. This creates a a file called "yourmapname.rsf".  Now open my level tester and go to File >> Then go to Open Script and go to the location of where the .rsf you dumped is. Then the map should load.

Once the map is loaded you can save the map as a "Quick Map".  A quick map allows you to load a map faster and easier.  This is best used on large maps that take a long time to load.

 

2. Tips
