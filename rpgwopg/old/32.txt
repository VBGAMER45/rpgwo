Ok well I was spitting out ideas in game so James gave me the brilliant idea.. TO POST HERE! DA.. ::Bats himself:: Hehe. Anyway here it goes. Alot of us have been complaining about the lack of trust and ability in an admin. The term admin usually defines someone who is high above the others running the, game, buisness, organization, website and so on. If we are to run the game we need a better system. First off Im introducing the idea of Admin Levels. Basically a 3-4 structure system which limits the Admins power or heightens it.<br><br>
 So my idea would be... When you first get admin access you are not as trusted. Your level would be that of a Helper. Just a person who's job is to help the players. You would have small commands, such as pk make, warp and a quest maker. Basically make about 20 premade quests in a txt file and give them that, which is what there alloud to use. Then from there you go to Moderator. This would be the current admins now basically. Except there commands would not include, ban, and MAY limit some item getting and other things. After you would become lvl3 which is Gm or Game Master. Or you can call it whatever. This would be our current admins, with a few additions or whatnot. Now the highest lvl would be Admin. With the ability to do all we can do now, edit player stats, game functions such as time, land, etc. Access to the UPDATER. And even minor coding development. This would be the max. People who are trusted enough to run the game and help with it. Anyway these are just <b>my ideas</b> but I think you guys will agree with some of it. Gimme any suggestions you want. Bya! - Jay :D
